# Data-Driven Scheduling & Risk Analysis

This project focuses on modeling and optimizing a procurement process under uncertainty using analytical and simulation-based approaches.

## Objectives
- Model project scheduling using deterministic and probabilistic methods
- Analyze risk in project timelines
- Improve decision-making under constraints

## Methods
- Critical Path Method (CPM) for baseline scheduling
- Monte Carlo simulation for probabilistic risk analysis
- Constraint-based optimization using Z3 solver

## Tools
- Python
- NumPy, Pandas
- Simulation techniques
- Z3 SMT Solver

## Key Insights
- Deterministic schedules may underestimate project risk
- Simulation provides a more realistic view of completion times
- Constraint-based optimization improves feasibility under real-world limitations
